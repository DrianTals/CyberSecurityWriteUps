# Day 1 - Web-Client Challenge

### 1st Challenge: HTML - disabled buttons (Easy Mode)  

1. Simply right click the disabled button

2. Click Inspect

3. edit the html source code and delete/erase the "disable" in the button code

4. Input the text "auth-login" and hit enter

5. you get the flag/validation password completing the challange

### 2nd Challenge: Javascript - Authentication (Easy Mode)

1. open the link "http://challenge01.root-me.org/web-client/ch9/"

2. Go the Developer Tools or Right Click and Inspect the page

3. go to the Network Tab, and click login.js file

4. and user and password would appear like this and use the password to validate completion of the challenge

### 3rd Challenge: Javascript - Source (Easy Mode) 

1. Open the link "http://challenge01.root-me.org/web-client/ch1/"

2. The page ask for the password, you can try and guess the password.

3. Click Cancel and right click the page and click "View Page Source" or Ctrl+U

4. and this source will appear with its password and use the pass to validate complete the challenge
