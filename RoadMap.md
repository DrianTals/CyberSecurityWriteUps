
# 🛣️ February Roadmap: Web Exploitation (Beginner)

## 🎯 Goal for February:

Understand basics of web hacking and solve beginner labs (TryHackMe, PortSwigger, Root-Me)

---

# 🗓️ Week 1 (Feb 1–7) — Web Basics + Setup

*(If you already know some, just review quickly)*

### 📚 Learn:

* How websites work (HTTP/HTTPS)
* Client vs Server
* Request & Response
* URL, cookies, sessions
* Basic HTML, JS, forms

### 🛠️ Install tools:

* Google Chrome/Firefox
* **Burp Suite Community**
* VS Code
* TryHackMe account

### 🧪 Practice:

* TryHackMe: **Pre Security path**
* Learn how to inspect element (F12)
* Learn how login forms work

---

# 🗓️ Week 2 (Feb 8–14) — Web Exploitation Fundamentals

### 📚 Study:

* What is Web Exploitation
* How hackers find vulnerabilities
* HTTP methods (GET, POST)
* Parameters (id=1, username=admin)

### 🔥 Focus vulnerabilities:

1. **HTML Injection**
2. **Basic XSS (Cross-site scripting)**
3. **View source trick**

### 🧪 Practice labs:

* TryHackMe: Web Fundamentals
* PortSwigger: XSS labs (beginner)
* Root-Me: HTML & JS challenges

### 🎯 Goal:

Understand how input boxes can be exploited

---

# 🗓️ Week 3 (Feb 15–21) — Real Web Attacks

### 📚 Learn:

1. **SQL Injection (IMPORTANT)**
2. Login bypass
3. Basic authentication bypass
4. Cookies & sessions

### 🛠️ Tools:

* Burp Suite (intercept request)
* Browser DevTools

### 🧪 Practice:

* TryHackMe: SQL Injection room
* PortSwigger: SQLi labs
* Root-Me: SQL injection basic

### 🎯 Goal:

Login bypass using `' OR 1=1 --`

# 🗓️ Week 4 (Feb 22–28) — Beginner Hacker Skills

### 📚 Learn:

* File upload vulnerability
* Directory brute force
* Inspect hidden pages
* Basic recon (finding admin page)

### 🛠️ Tools to try:

* Burp Suite
* dirsearch (optional)
* gobuster (optional)

### 🧪 Practice:

* TryHackMe: Web rooms
* Root-Me challenges
* Beginner CTF web challenges

### 🎯 Goal:

Solve at least **3–5 web challenges**
